# PocketCode 🚀

PocketCode is a mobile code editor app designed to let developers write, edit, and manage code directly from their phones.  

## ✨ Features
- Lightweight and easy to use  
- Syntax highlighting (planned)  
- Save and manage multiple files  
- Portable code editor on the go
